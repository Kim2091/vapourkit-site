> Contribution guidelines are evolving. For now, the best paths are:

## Reporting bugs

Open an issue on [GitHub](https://github.com/Kim2091/vapourkit/issues) with:

- Your OS, GPU, and driver version
- The exact steps to reproduce
- Any relevant log output (Vapourkit writes to `main.log` in the install directory)

## Pull requests

PRs are welcome on [GitHub](https://github.com/Kim2091/vapourkit). For non-trivial changes, open an issue or post in the [Discord](https://discord.gg/uYKMn2hGwB) first so we can discuss approach.

## Filter templates

If you've made a useful filter template you'd like to share, drop the `.vkfilter` file into the Discord or open a PR to add it to the bundled set in `include/plugins/plugin_filters/`.
